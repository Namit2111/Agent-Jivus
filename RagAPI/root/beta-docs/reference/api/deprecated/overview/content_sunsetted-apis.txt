HubSpot APIs | Deprecated APIs
Sunsetted and deprecated APIs
The APIs listed in the sections below will
not
receive future updates to functionality, and in some cases, will be fully sunsetted and unavailable for use in your integrations.
It's highly recommended that you subscribe to the
HubSpot Developer Changelog
to follow along with the latest updates, breaking changes, and other significant changes to functionality.
Sunsetted APIs
The following APIs have been fully removed, and will return an error when making a call to their associated endpoints.
